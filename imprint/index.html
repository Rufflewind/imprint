Crate imprint [−] [src]

Provides the ability to imprint values at the type level, enabling compile-time validation of values that only exist at run time.

Heavily inspired by Richard Eisenberg's singletons library, Edward Kmett's reflection and eq libraries, as well as Gankro's sound unchecked indexing approach.
